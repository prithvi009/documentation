## Installation

### Script Tag (Recommended)

Add this single line to your website's `<head>`:

```html
<script src="https://events.tracksource.io/sdk/ts.min.js" data-tenant="YOUR_TENANT_ID"></script>
```

That's it! The SDK automatically:
- Initializes with your tenant ID
- Tracks page views
- Captures UTM parameters, fbclid, gclid
- Creates anonymous visitor IDs
- Stores attribution data in `window.tsAttribution`

### NPM Package (Coming Soon)

For React/Next.js projects, wrap the script in a component:

```jsx
// components/TrackSource.jsx
import { useEffect } from 'react';

export function TrackSource({ tenantId }) {
  useEffect(() => {
    if (window.TracksSource) return;
    const script = document.createElement('script');
    script.src = 'https://events.tracksource.io/sdk/ts.min.js';
    script.setAttribute('data-tenant', tenantId);
    script.async = true;
    document.head.appendChild(script);
  }, [tenantId]);
  return null;
}
```

## Configuration Options

The SDK auto-configures from the `data-tenant` attribute. For advanced configuration, use `data-*` attributes:

```html
<script
  src="https://events.tracksource.io/sdk/ts.min.js"
  data-tenant="YOUR_TENANT_ID"
  data-debug="true"
></script>
```

| Attribute | Default | Description |
|-----------|---------|-------------|
| `data-tenant` | Required | Your tenant ID |
| `data-debug` | `false` | Enable console logging |

## Core Methods

The SDK exposes `window.TracksSource` with the following methods:

### Track Events

```javascript
// Basic event
TracksSource.track('button_click');

// Event with properties
TracksSource.track('signup', {
  plan: 'pro',
  source: 'pricing_page'
});

// Conversion event with value
TracksSource.track('conversion', {
  email: 'customer@example.com',
  conversion_value: 99.00,
  currency: 'USD',
  order_id: 'ORD-123'
});
```

### Identify Users

```javascript
// Identify with email
TracksSource.identify('john@example.com');

// Identify with email and properties
TracksSource.identify('john@example.com', {
  name: 'John Doe',
  phone: '+1-555-123-4567',
  company: 'Acme Inc',
  plan: 'enterprise'
});
```

### Page Views

Page views are automatically tracked. For SPAs, manually track route changes:

```javascript
// Manual page view for SPA route changes
TracksSource.track('page_view', {
  path: '/new-page',
  title: 'New Page Title'
});
```

### Get Attribution Data

Attribution data is stored in `window.tsAttribution`:

```javascript
// Access attribution data directly
const data = window.tsAttribution || {};

console.log(data);
// {
//   anonymous_id: "anon_abc123",
//   utm_source: "facebook",
//   utm_medium: "paid",
//   utm_campaign: "spring_sale",
//   fbp: "fb.1.1234567890.987654321",
//   fbc: "fb.1.1234567890.IwAR3...",
//   gclid: "Cj0KCQiA...",
//   landing_page: "/pricing",
//   referrer: "https://facebook.com"
// }
```

## Standard Events

Use these standard event names for best compatibility with ad platforms:

| Event | Description | Required Properties |
|-------|-------------|---------------------|
| `page_view` | Page was viewed | Auto-tracked |
| `lead` | Lead form submitted | `email` |
| `signup` | User signed up | `email` |
| `login` | User logged in | `email` |
| `purchase` | Purchase completed | `email`, `revenue`, `currency` |
| `add_to_cart` | Item added to cart | `product_id`, `price` |
| `checkout_started` | Checkout initiated | `cart_value` |

### Purchase Event Example

```javascript
TracksSource.track('conversion', {
  // Required
  email: 'customer@example.com',
  conversion_value: 149.00,
  currency: 'USD',

  // Recommended
  order_id: 'ORD-12345',
  product_id: 'PROD-001',
  product_name: 'Pro Plan Annual',
  quantity: 1,

  // Optional
  coupon: 'SAVE20',
  discount: 30.00,
  tax: 12.50,
  shipping: 0
});
```

## Form Integration

### Hidden Fields for Webhooks

Add hidden fields to pass attribution data through form submissions:

```html
<form id="contact-form">
  <input type="email" name="email" required>
  <input type="text" name="name" required>

  <!-- Hidden attribution fields -->
  <input type="hidden" name="ts_anonymous_id">
  <input type="hidden" name="utm_source">
  <input type="hidden" name="utm_medium">
  <input type="hidden" name="utm_campaign">
  <input type="hidden" name="fbp">
  <input type="hidden" name="fbc">
  <input type="hidden" name="gclid">

  <button type="submit">Submit</button>
</form>

<script>
  // Auto-populate from window.tsAttribution
  document.addEventListener('DOMContentLoaded', function() {
    var data = window.tsAttribution || {};
    var fields = ['ts_anonymous_id', 'utm_source', 'utm_medium', 'utm_campaign', 'fbp', 'fbc', 'gclid'];

    fields.forEach(function(field) {
      var input = document.querySelector('[name="' + field + '"]');
      var key = field === 'ts_anonymous_id' ? 'anonymous_id' : field;
      if (input && data[key]) {
        input.value = data[key];
      }
    });
  });
</script>
```

## SPA Support

For single-page applications (React, Vue, Next.js), track route changes:

```javascript
// React Router example
import { useEffect } from 'react';
import { useLocation } from 'react-router-dom';

function App() {
  const location = useLocation();

  useEffect(() => {
    if (window.TracksSource) {
      TracksSource.track('page_view', {
        path: location.pathname
      });
    }
  }, [location]);

  return <Routes>...</Routes>;
}
```

```javascript
// Vue Router example
router.afterEach((to) => {
  if (window.TracksSource) {
    TracksSource.track('page_view', {
      path: to.path,
      name: to.name
    });
  }
});
```

## Debug Mode

Enable debug mode to see events in the console:

```html
<script src="https://events.tracksource.io/sdk/ts.min.js" data-tenant="YOUR_TENANT_ID" data-debug="true"></script>
```

This will log all events and API responses to the browser console.
